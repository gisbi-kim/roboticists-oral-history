# Gerd Hirzinger — 행적 연보

> 출처: ETHW 구술 인터뷰 (`ethw_source_eng/hirzinger.md`)

```
1945           — 독일 Bavaria 주 Schwandorf 출생
유년·학창기     — Bavaria 북부 Amberg에서 학창 생활
1969           — Technical University of Munich 전자공학 학위 (Dipl.-Ing.) 취득; DLR 입사 — 처음에는 간섭계·고주파 기술
DLR 초기        — 6개월 후 제어 이론으로 전환; 디지털 제어 PhD 시작 (PDP-11 등 process computer); 위성 자세 제어, 공기 베어링 테이블, 불안정 항공기 (CCV) 최적 디지털 제어 연구
1974           — TU Munich에서 PhD 취득 — 디지털 제어
1976           — DLR Automation 그룹 리더 취임
1976~1977경   — 디지털 제어 기계 시스템 응용 브레인스토밍 — 매니퓰레이터·수중·의수족 검토; BMW의 Wolfgang Reitzle 방문 후 ABB 로봇 1대 기증받음
1978           — Hannover Fair에서 첫 force-torque 센서로 BMW 오일펌프 자동 조립 시연
1970년대 후반   — 첫 force-torque 센서 개발 (유럽 최초로 추정); 좌표 변환을 4-bit 마이크로컨트롤러에 구현
1982           — 두 번째 ABB 로봇 구입; 두 로봇 협조 제어, control ball 인터페이스 개발
1985           — IEEE St. Louis 학회에서 발표 → Mike Brady 등에게 인지; 첫 ISRR (Goviller, France) 초청; Oussama Khatib와 첫 만남 — torque control 영감
1987경         — Spacelab D2 셔틀 미션 ROTEX 로봇 실험 제안 수락; lightweight robot 구상 시작
1991           — TU Munich 교수직 겸임
1991/1992      — 첫 lightweight robot 완성 — 우주비행사 훈련용
1992           — Institute of Robotics and Mechatronics Center (RMC) 디렉터 취임
1993년 4월      — Spacelab D2 미션 발사, ROTEX 실험 — 6~7초 지연 원격조작, 자율 공중 부양 물체 파지 성공; ROTEX 직후 Leibniz Award 수상 (150만 유로) → 의료 로봇 연구 시작
1994           — KUKA와 본격 협력 — 실시간 로봇 동역학 모델 제공 (BMW 등에서 채택)
1990년대 중후반— RoboDrive 모터 시리즈 시뮬레이션 최적화 (약 2년)
1990년대 후반   — KUKA가 BMW·Mercedes·Ford 경쟁 수주 (Hirzinger 그룹 기여); SensoDrive·Dualis 등 spin-off 설립; Logitech이 Space Mouse 인수
1997           — 일본 ETS-VII 자유비행 로봇 발사 (Dr. Oda 팀)
1999           — 일본 Tsukuba 방문, 지상에서 우주 로봇 ETS-VII 원격 제어; 위성 자세 재배향 실험
2000년대 초     — 3세대 lightweight robot 완성 (carbon fiber, RoboDrive 모터)
2005경         — 우주 정거장 외측 ROKVISS 실험 시작 — 2개 관절, 20ms 지연 텔레프레전스
2010           — IEEE Fellow
2011년 5월      — ROKVISS 시스템 우주에서 회수 (카자흐스탄 착륙); 6년 우주 노출 분석 예정
2011년 7월 20일 — Peter Asaro 인터뷰 (Wessling, Germany)
지속 연구       — MiroSurge 의료 로봇, ATHLETE 협력 (JPL), Mars/달 rover (Part-Time Scientists 팀), 인공심장 (Dualis), 가변 강성 hand-arm 시스템
```
